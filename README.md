# 🎮 Modern Minecraft Launcher

Сучасний, кросплатформений лаунчер для гри Minecraft, написаний на Python. 
Головна мета проекту — надати користувачам швидкий, надійний та зручний інструмент для керування різними версіями та інстансами Minecraft, з автоматичним вирішенням проблем із сумісністю Java.

## ✨ Основні функції

- **Управління інстансами:** Створення ізольованих інстансів (Vanilla, Forge, Fabric). Кожен інстанс має власні моди, збереження та налаштування.
- **Динамічне завантаження Java:** Лаунчер автоматично визначає потрібну версію Java (8, 17, 21) і завантажує офіційний дистрибутив Adoptium (Eclipse Temurin), якщо її немає в системі.
- **Інтеграція з Mojang:** Динамічне завантаження списку версій та файлів гри безпосередньо з офіційних серверів.
- **Сучасний інтерфейс:** Красивий темний дизайн (Catppuccin Mocha) на базі PySide6, який не зависає під час завантаження завдяки багатопоточності (QThread).
- **Індикація статусу:** Візуальне відображення того, які версії вже завантажено, а які потрібно встановити.
- **Офлайн-режим:** Гнучке налаштування нікнейму та виділення оперативної пам'яті (RAM) для гри.

## 🛠 Технології

- [Python 3](https://www.python.org/) - Мова програмування
- [PySide6](https://doc.qt.io/qtforpython-6/) - Графічний інтерфейс (Qt for Python)
- [minecraft-launcher-lib](https://minecraft-launcher-lib.readthedocs.io/) - Ядро для взаємодії з файлами Minecraft
- [PyInstaller](https://pyinstaller.org/) - Упакування в виконуваний файл (.exe)

## 🚀 Встановлення та запуск (для розробників)

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/ВАШ_ПРОФІЛЬ/ВАШ_РЕПОЗИТОРІЙ.git
   cd ВАШ_РЕПОЗИТОРІЙ
   ```

2. Створіть та активуйте віртуальне середовище (рекомендовано):
   ```bash
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   # Linux:
   source venv/bin/activate
   ```

3. Встановіть залежності:
   ```bash
   pip install PySide6 minecraft-launcher-lib requests
   ```

4. Запустіть лаунчер:
   ```bash
   python main.py
   ```

## 📦 Компіляція в .exe (Windows)

Для створення одного виконуваного файлу з іконкою та метаданими використовуйте PyInstaller:

1. Встановіть PyInstaller:
   ```bash
   pip install pyinstaller
   ```

2. Переконайтеся, що у корені проекту є файли `icon.ico` та `version.txt`.

3. Виконайте команду збірки:
   ```bash
   pyinstaller --name "ModernMCLauncher" --onefile --windowed --icon="icon.ico" --version-file="version.txt" --collect-data PySide6 main.py
   ```

4. Готовий файл з'явиться у папці `dist/ModernMCLauncher.exe`.

## 📂 Де зберігаються дані?

Щоб не засмічувати папку з програмою, лаунчер зберігає всі дані (інстанси, бібліотеки, Java, налаштування) у стандартній директорії ОС:

- **Windows:** `C:\Користувач\ВашПрофіль\AppData\Roaming\ModernMCLauncher`
- **Linux:** `~/.local/share/ModernMCLauncher`
